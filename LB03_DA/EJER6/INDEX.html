Ejercicio 6 de Grid Responsive

Este ejercicio muestra cómo usar CSS Grid para crear un diseño adaptable. En pantallas grandes (escritorio) se usa una distribución compleja con grid-template-areas que forma un mosaico de 7 bloques diferentes. En pantallas pequeñas (móviles), mediante un breakpoint, la grilla se simplifica a 2 columnas, lo que permite que el contenido fluya de manera más legible. Cada bloque tiene un borde negro para diferenciar claramente las áreas.
